# Blackbird Prototype

Interactive HTML prototype for the Blackbird policy platform. All data is sample data.

**Live site:** connect this repo to Netlify or Cloudflare Pages (no build command, publish directory `/`).

## Current scope
- Policies list (search, filters, saved views, sorting, pagination)
- Policy Detail: Policy Info, Rating quotes, Documents, Claims, Loss Runs, UDocs
- Policy Preparation workspace: Rating, Forms (list + period comparison), Subjectivities
- Utility rail panels: Policyholder, Documents, Notes, Subjectivities, Tasks, History, AI Bot
- Sticky compact policy header on scroll

Updated automatically by Claude as designs evolve. Each commit message describes the change.
